# WebSocket Demo - Unity + Python

## Requisiti

### Python
- Python 3.7 o superiore
- Libreria websockets: `pip install websockets`

### Unity
- Unity 2020.3 o superiore
- NativeWebSocket package

## Istruzioni

### 1. Avvia il server Python
```bash
cd serverWSPython
python server_websocket.py
```

### 2. Apri il progetto Unity
- Apri Unity Hub
- Aggiungi il progetto dalla cartella `unity-wsClient`
- Apri la scena principale
- Premi Play
